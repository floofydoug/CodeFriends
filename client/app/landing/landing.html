Collaborate on Code in Real Time

CodeFriends is Google Docs for Sublime Text. Work on code in the same way you always do, but with a toolset that lets you collaborate with anyone around the world.

 

Real Time Editing

Working real-time on projects means that you can always have the same code your collaborators have, down to the millisecond. Projects also get magically updated as new files are created and deleted.

Clone From Git

Use the code you already have by importing it directly from GitHub into a CodeFriends project. Don't interrupt your normal workflow. Use the tools that you know and love!

Video and Chat work together remotely

Real Time Communication using WebRTC protocols lets you setup Video Chat with your remote peers without any extra plugins. Persistant chat keeps your important conversations saved so that you won't miss important messages.

Run Your Code

Code with a friend, then run it straight in the browser. Output is logged onto your browser client. CodeFriends compiles 12 different languages: C#, C/C++, Clojure, Java, Go, JavaScript, PHP, Python, Ruby, Scala, VB.NET, Bash, and Objective-C. The syntax highlighter currently supports HTML, CSS, Javascript, PHP, and Ruby.

Upload And Download Your Projects

Take your project where ever you want by downloading your project off of CodeFriends. Import complete projects using .zip uploads for projects and uploading individual files to a project.
